
## SMSForBets.com Campaign Bot for progress automation

### Installation
- Node.JS v8
- Project dependencies: `npm install`

### Config
- `config.yaml`

### Screenshots
- `images` in the root folder

### Logs
- `data/logs` in the root folder

### Run

#### Run with node
```shell
npm run build
npm start
```

#### Run with pm2
```
npx pm2 start npm --name 'SMSForBets.com' -- start
```
